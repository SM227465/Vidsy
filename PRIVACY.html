Vidsy — Privacy Policy

Last updated: 2026-04-19

Vidsy is a Chrome extension that detects and downloads video and audio streams from web pages. This policy describes what data Vidsy handles and what it does not do.

Data Vidsy does not collect

Vidsy does not collect, transmit, sell, or share any personal information. Specifically, Vidsy does not collect:

  • Personally identifiable information (name, email, address, phone number, identifiers)
  • Health information
  • Financial or payment information
  • Authentication credentials
  • Personal communications
  • Location
  • Browsing or web history
  • Clicks, mouse movements, or other user activity
  • Page content beyond the metadata required to name a download (title, thumbnail, canonical URL)

Data Vidsy stores locally on your device

Vidsy uses chrome.storage.local to persist the following on your machine only. This data never leaves your browser:

  • User preferences (theme, filename template, history retention limit, merge toggles)
  • Download history (title, source URL, file size, timestamp) — retention is user-configurable and can be cleared at any time from the extension options page

During active downloads, Vidsy writes segment data to the Origin Private File System (OPFS) as scratch space for local WebAssembly remuxing. These scratch files are deleted as soon as the download completes or is cancelled.

Remote servers

Vidsy does not communicate with any server operated by the developer. The only network requests Vidsy makes are:

  • Fetches for the video and audio streams the user explicitly chose to download, sent directly to the origin servers hosting that media
  • Fetches for page metadata (title, og:image) on the page the user is currently viewing

No analytics, telemetry, crash reporting, or third-party tracking is included.

Permissions

Vidsy's Chrome permissions are documented in the Chrome Web Store listing. Each is scoped to the extension's single purpose: detecting media on the current page and saving downloads to the user's Downloads folder. No permission is used to collect or transmit user data.

Children

Vidsy is not directed at children under 13 and does not knowingly collect data from anyone.

Changes

If this policy changes, the "Last updated" date at the top will change. Material changes will be noted in the extension's release notes.
